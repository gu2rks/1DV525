What is Robots.txt file?

The robots.txt file is a simple text file placed on your web server which tells webcrawlers (Googlebot) if they should access a file or not.

I have a robots.txt at my root web directory (https://1dv525.github.io/aa224iu-examination-1). I block Googlebot to access to contact.html as the assignment required. Also, I chose to block the bot to access my pictures by Disallow: /images/ , which mean block Google bot to access images directory + all children file/directory that exits inside the"images" directory Why? because why not? It is my images anyway :P

But wait, Googlebot is not the only webcrawler that exits on the internet. Then I decided to create another group and as a User-agent: *, which mean block all other webcrawler to access images directory too

IOS users have been hack for years by visiting a hecked website

Hacker use vulnerability in the Safari web-browser to gain root access

Early this year, a Google researcher explained how some of the IOS users might have been hack by just visit a hacked site. Hacker uses the vulnerability in the Safari web-browser to gain root access to their devices.

The iPhone exploits were used to deploy and implant (spyware) which are design to steal files such as iMessages, photos, and live GPS location data of users, and upload them to an external server every 60 seconds. The implant also had access to the victim's device's keychain data containing credentials, authentication tokens, and certificates used on and by the device.

"The best part" is the implant will automatically wipe off from an infected iPhone upon rebooting and leaving no trace of itself. The implant will be reinstalled only if the user visiting the hacked site again.
